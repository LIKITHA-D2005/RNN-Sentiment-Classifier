# 💬 Intelligent Sentiment Classifier: RNN-Based Sentiment Analysis

## 📌 Overview
This project implements a **Recurrent Neural Network (RNN)** for sentiment analysis on textual data.

The model learns sequential patterns in text to classify sentiments (positive/negative), demonstrating the effectiveness of deep learning in Natural Language Processing (NLP) tasks.

---

## ⚙️ Model Architecture

### 🔹 Recurrent Neural Network (RNN)
• Processes sequential text data effectively  
• Captures contextual dependencies between words  
• Learns sentiment patterns from sequences  

### 🔹 Text Processing
• Tokenization and sequence conversion  
• Padding for uniform input length  
• Vocabulary encoding for model input
